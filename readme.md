🚌 **P.H Paribahan - Bus Ticket Booking Interface**
P.H Paribahan is a sleek, responsive bus ticketing landing page designed to provide a seamless booking experience for travelers. Built using a modern frontend stack of Tailwind CSS and JavaScript, the project focuses on high-speed performance, clean aesthetics, and intuitive navigation.

🌟 Key Features
Dynamic Seat Selection: An interactive JavaScript-powered bus seating layout where users can select seats in real-time with automatic price calculation.

Utility-First Design: Styled entirely with Tailwind CSS for a professional, mobile-first look that ensures a consistent experience across all devices.

Real-time Validation: Integrated form validation for passenger details and coupon codes using Vanilla JavaScript.

Automated Ticketing: A dynamic receipt generation system that updates the total cost and seat count as users interact with the interface.

Responsive Layout: Optimized grid and flexbox structures for smooth browsing on mobile, tablet, and desktop.

💻 Technology Stack
Structure: HTML5

Styling: Tailwind CSS (Utility-first framework)

Interactivity: Vanilla JavaScript (ES6+)

Icons/Fonts: Google Fonts & FontAwesome
